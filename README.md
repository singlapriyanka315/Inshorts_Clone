# Inshorts_Clone
===============================

## About the Project
This is a MERN project clone of a short news application.

- Built using Front-end: HTML, CSS, JavaScript, Material UI, React.
              Back-end : NODE, Mongodb, Express.
              
- User-friendly interface
- Responsive for all screen size.
